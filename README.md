# Dynamic System Modeling & Stability Analysis
**Munich University of Applied Sciences**

* **Simulated** complex physical systems (Harmonic Oscillators & Chemical Reaction Networks) by constructing and solving State-Space representations.
* **Performed** Eigenvalue/Eigenvector Analysis to characterize system behavior, distinguishing between underdamped, overdamped, and critically damped regimes.
* **Investigated** Numerical Stability in stiff ODE systems, identifying how floating-point perturbations and matrix defectiveness impact simulation robustness.
* **Modeled** multi-stage chemical kinetics, analyzing concentration profiles and steady-state solutions through linear chain reaction networks.
* **Conducted** Parameter Studies using Matplotlib to visualize phase-space transitions and amplitude decay.

### Tech Stack
* **Language:** Python
* **Libraries:** SciPy (`odeint`, `solve_ivp`), NumPy, Matplotlib
* **Theory:** Linear Systems Theory, ODE Stability
